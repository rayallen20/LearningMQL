# 5. 时间函数

- `Day()`: 返回值表示当前服务器时间的日,类型为int
- `DayOfWeek()`: 返回值表示当前服务器时间的星期,类型为int
- `Year()`: 返回值表示当前服务器时间的年,类型为int
- `Month()`: 返回值表示当前服务器时间的月,类型为int
- `Hour()`: 返回值表示当前服务器时间的小时,类型为int
- `Minute()`: 返回值表示当前服务器时间的分钟,类型为int
- `Second()`: 返回值表示当前服务器时间的秒,类型为int
- `TimeLocal()`: 返回值表示本地时间,类型为datetime
  - 注意: 平台商服务器时间和北京时间不同,存在时差

```mq4
//+------------------------------------------------------------------+
//| Expert tick function                                             |
//+------------------------------------------------------------------+
void OnTick()
  {
//---
   PrintDay();
   PrintDayOfWeek();
   PrintYear();
   PrintMonth();
   PrintHour();
   PrintMinute();
   PrintTimeLocal();
   PrintTimeCurrent();
  }
  
//+------------------------------------------------------------------+
void PrintDay() {
   int myDay = Day();
   string myDayStr = IntegerToString(myDay);
   Print("今天的日期是当月的第: " + myDayStr + " 天");
}

void PrintDayOfWeek() {
   int myDayOfWeek = DayOfWeek();
   string myDayOfWeekStr = IntegerToString(myDayOfWeek);
   Print("今天是本周的第: " + myDayOfWeekStr + " 天");
}

void PrintYear() {
   int myYear = Year();
   string myYearStr = IntegerToString(myYear);
   Print("今年为: " + myYearStr + " 年");
}

void PrintMonth() {
   int myMonth = Month();
   string myMonthStr = IntegerToString(myMonth);
   Print("当前月份为: " + myMonthStr);
}

void PrintHour() {
   int myHour = Hour();
   string myHourStr = IntegerToString(myHour);
   Print("当前小时为: " + myHourStr);
}

void PrintMinute() {
   int myMinute = Minute();
   string myMinuteStr = IntegerToString(myMinute);
   Print("当前分钟为: " + myMinuteStr);
}

void PrintTimeLocal() {
   datetime myTimeLocal = TimeLocal();
   string myTimeLocalStr = TimeToString(myTimeLocal, TIME_DATE|TIME_SECONDS);
   Print("当前本地时间为: " + myTimeLocalStr);
}

void PrintTimeCurrent() {
   datetime myTimeCurrent = TimeCurrent();
   string myTimeCurrentStr = TimeToString(myTimeCurrent, TIME_DATE|TIME_SECONDS);
   Print("当前平台服务器时间为: " + myTimeCurrentStr);
}
```

注: `TimeCurrent()`返回的时间比本地时间早了7个小时,即`UTC +1`时区.是欧洲中部时间(北京是`UTC +8`时区),猜测平台商的服务器是在欧洲的.
