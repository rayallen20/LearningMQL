# 1. `High[i]`

- `High[i]`: `i`为K线的序号(和数组下标的概念类似).表示第`i`根K线的最高价.

```mq4
//+------------------------------------------------------------------+
//| Expert tick function                                             |
//+------------------------------------------------------------------+
void OnTick()
  {
//---
   Print(High[0]);
  }
```
