# 1. `Time[i]`

- `Time[i]`: `i`为K线的序号(和数组下标的概念类似).表示第`i`根K线的开盘时间.
  
```mq4
//+------------------------------------------------------------------+
//| Expert tick function                                             |
//+------------------------------------------------------------------+
void OnTick()
  {
//---
   Print(Time[0]);
  }
```
