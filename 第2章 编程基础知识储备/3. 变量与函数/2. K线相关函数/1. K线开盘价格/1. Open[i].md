# 1. `Open[i]`

- `Open[i]`: `i`为K线的序号(和数组下标的概念类似).表示第`i`根K线的开盘价.

```mq4
//+------------------------------------------------------------------+
//| Timer function                                                   |
//+------------------------------------------------------------------+
void OnTimer()
  {
//---
   Print(Open[0]);
  }
//+------------------------------------------------------------------+
```
