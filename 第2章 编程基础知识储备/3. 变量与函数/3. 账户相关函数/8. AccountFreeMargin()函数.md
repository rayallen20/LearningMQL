# 8. `AccountFreeMargin()`函数

- `AccountFreeMargin()`: 返回值表示账户可用保证金,类型为double

```mq4

//| Expert tick function                                             |
//+------------------------------------------------------------------+

  {
//---
   double myAccountFreeMargin = AccountFreeMargin();

  }
```
