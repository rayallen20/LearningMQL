# 5. `AccountLeverage()`函数

- `AccountLeverage()`: 返回值表示杠杆比例,类型为int

```mq4
//+------------------------------------------------------------------+
//| Expert tick function                                             |
//+------------------------------------------------------------------+
void OnTick()
  {
//---
   int myAccountLeverage = AccountLeverage();
   Print(myAccountLeverage);
  }
```

Tips: 模拟账户默认杠杆为500倍,我猜测这个是可以调的
