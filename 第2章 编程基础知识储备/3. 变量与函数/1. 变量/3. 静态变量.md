# 3. 静态变量

静态变量被存放在内存静态存储区里,在函数运行结束后静态变量的值不会丢失.若静态变量没有明确的初始化值,它将被初始化为零.
静态变量只可以在初始加载函数之前初始化1次.**当从定义了静态变量的函数内部退出时,静态变量值不会丢失**.

有点像PHP里的`static`变量.

```mq4
//+------------------------------------------------------------------+
//| Expert initialization function                                   |
//+------------------------------------------------------------------+
int OnInit()
  {
   PrintCounter();
   PrintCounter();
   PrintCounter();
//--- create timer
   EventSetTimer(60);
   
//---
   return(INIT_SUCCEEDED);
  }

void PrintCounter()
  {
    static int counter = 0;
    counter++;
    Print(counter);
  }
```
