# 4. `bool`

布尔型(Boolean)变量,用来表示:

- 真值(`true`)
- 假值(`false`)

还可以用数字`1`和`0`表示

```mq4
//+------------------------------------------------------------------+
//| Expert tick function                                             |
//+------------------------------------------------------------------+
void OnTick()
  {
//---
   bool myBool = false;
   Print(myBool);
  }
```
