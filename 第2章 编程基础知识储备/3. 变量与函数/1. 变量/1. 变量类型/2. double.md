# 2. `double`

由:

- 整数部分
- 小数点(`.`)
- 小数部分

组成.其中整数部分和小数部分都是一组十进制数字.

```mq4
//+------------------------------------------------------------------+
//| Expert tick function                                             |
//+------------------------------------------------------------------+
void OnTick()
  {
//---
   double myDouble = 1.23;
   Print(myDouble);
  }
```
