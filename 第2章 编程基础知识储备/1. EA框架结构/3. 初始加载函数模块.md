# 3. 初始加载函数模块

该模块主要用于**加载EA时运行,且只运行一次**(有点像GO的`init()`函数).

有些策略意图(例如:在加载EA时核对账户号码是否被允许运行)是不需要重复执行,只需要判断1次即可的.则这种逻辑就可以放在该模块中.
**该模块中的代码只在EA加载时运行一次**,这是区别于主程序模块很重要的一点.

```mq4
//+------------------------------------------------------------------+
//| Expert initialization function                                   |
//+------------------------------------------------------------------+
int OnInit()
  {
//--- create timer
   EventSetTimer(60);
   
//---
   return(INIT_SUCCEEDED);
  }
```
