# 1. `OnTimer()`函数

在使用该函数时需要:

- 在EA初始加载函数模块中,为`OnTimer()`设置一个运行间隔时间
- 在EA退出加载函数模块中,删除给定的运行间隔时间

```mq4
//+------------------------------------------------------------------+
//| Expert initialization function                                   |
//+------------------------------------------------------------------+
int OnInit()
  {
//--- create timer
   EventSetTimer(60);
   
//---
   return(INIT_SUCCEEDED);
  }
//+------------------------------------------------------------------+
//| Expert deinitialization function                                 |
//+------------------------------------------------------------------+
void OnDeinit(const int reason)
  {
//--- destroy timer
   EventKillTimer();
   
  }
```

- `EventSetTimer(K)`: 指定函数`OnTimer()`设置运行间隔时间,其中`K`为间隔的秒数
